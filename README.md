# 💱 Currency Exchange App

A beginner-friendly project that lets users convert currencies using real-time exchange rates 🌍💰

---

## 🚀 Features
- Convert between different currencies  
- Get live exchange rates using an API  
- Simple and clean interface  
- Works on desktop and mobile  

---

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  
- Exchange Rate API
